# ArabicDotter
 Build an NLP model for converting dotless Arabic text to dotted text

 ## Dotless special characters
| Symbol | Unicode Decimal |
|--------|------|
| ٮ     | U+0640 |
| ڡ      | 1697 |
| ٌ      | U+064C |
| ٍ      | U+064D |
| َ      | U+064E |
| ُ      | U+064F |
| ِ      | U+0650 |
| ّ      | U+0651 |
| ْ      | U+0652 |
